Flappy Bird Clone

2025

A recreation of the classic Flappy Bird game built in Unity. The game features physics-based movement, procedural obstacle spawning, collision detection, and responsive controls. This project demonstrates understanding of game loop structure, gameplay logic, and user input handling. This is my first published Unity project and part of a growing game development portfolio.

Core Skills

Unity Engine C# Programming Game Physics (Unity Rigidbody) Collision Detection Game Loop Logic Scene Management UI and Input Handling

Tech Stack

Unity C# Visual Studio WebGL Build Git / GitHub

What I Learned

While building this project, I learned how to implement physics-based player movement, manage game states such as start, play, and game over, spawn obstacles dynamically, and track player score. I also learned how to export a Unity project to WebGL and deploy it to GitHub Pages for online play.
